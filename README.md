# Biblioteca de Livros em Java

Um sistema simples de gerenciamento de biblioteca desenvolvido para exercitar conceitos de Orientação a Objetos em Java.

## 🚀 Funcionalidades
* Cadastro de livros (Título, Autor, Ano).
* Listagem de acervo via terminal.
* Gerenciamento através de listas dinâmicas (`ArrayList`).

## 🛠️ Tecnologias
* **Java 21**
* **VS Code**

## 📂 Estrutura do Projeto
* `src/model/Livro.java`: Representação da entidade Livro.
* `src/model/Biblioteca.java`: Lógica de gerenciamento do acervo.
* `src/Main.java`: Ponto de entrada e interface via console.
